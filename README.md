# Sales & Customer Dashboard for a Bike Company

## Project Overview

This project provides a Power BI dashboard for a bike company, featuring both sales and customer overviews. The dashboard enables users to filter data by month, year, or product category to uncover key business insights.

## Features

- **Sales Overview**: Track sales performance across different time periods and product categories.
- **Customer Overview**: Analyze customer demographics and behavior.
- **Flexible Filtering**: Filter dashboard data by month/year or product category.
- **Key Insights**: Discover patterns in pricing, occupancy, and customer reviews.
- **Investment-grade Insights**: Identify high-yield neighborhoods, seasonal demand trends, and performance metrics important to stakeholders.
- **Integrated Data Sources**: Utilizes SQL/Python for data extraction and aggregation from large datasets, streamlining reporting and improving data accessibility.

## Getting Started

Follow these steps to open and explore the Power BI project locally:

### 1. Clone the Repository

```bash
git clone https://github.com/JSubs/Data-Analyst-Portfolio.git
cd Data-Analyst-Portfolio
```

### 2. Install Power BI Desktop

Download and install [Power BI Desktop](https://powerbi.microsoft.com/desktop/) if you haven’t already.

### 3. Open the Power BI File

Locate the `.pbix` file in the project folder.

- Double-click it, or
- Open Power BI Desktop, then go to:  
  `File → Open → your_project_name.pbix`

### 4. Connect to the Data Source

If the report uses external data:
- Ensure the dataset (e.g., `.csv`, `.xlsx`, or database connection) is located in the specified folder or path.
- If paths differ, go to `Home → Transform data → Data source settings` and update the source path.

### 5. Refresh the Data

Click `Home → Refresh` to load the latest data into the report.

## Usage Examples
The Sales 2022 Dashboard provides an interactive overview of company sales performance across regions, products, and customers. Built in Power BI, this dashboard enables business managers, stakeholders, and sales teams to monitor key performance indicators and uncover actionable insights easily.

**Key Features**

• Regional Sales Analysis: Visualizes total revenue by state and region, enabling identification of top-performing areas and growth opportunities.

• Product Performance: Highlights best-selling products and categories, allowing for quick evaluation of product mix efficiency.

• Profit Trends: Tracks profit margins and growth trends over time, categorized by year and month.

• Customer Insights: Displays customer purchasing patterns and identifies high-value customers driving the majority of sales.

• Time-Based Exploration: Users can filter by year and month to analyze performance dynamics over specific periods.

**Main Insights**

Revenue distribution across regions and states

Monthly and yearly growth trends

Top-performing products and categories

Key customers contributing the highest revenue

**Purpose
**
This dashboard supports data-driven decision-making by providing a clear, visual summary of sales performance. It helps teams identify where to focus sales strategies, strengthen customer relationships, and optimize product portfolios.

**Tools Used**

Power BI for dashboard creation and interactive visualizations

Data Source: Sales data from 2022 (aggregated by region, product, and customer)
